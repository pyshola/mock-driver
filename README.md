Mock Driver

A simple Node.js Express application to simulate a mock driver.

## Installation

1.  Clone the repository or download the code.
2.  Navigate to the `mock-driver` directory.
3.  Install the dependencies:
    ```bash
    npm install
    ```

## Running the Application

1.  Start the server:
    ```bash
    node index.js
    ```
2.  The application will be running at http://localhost:3000.
